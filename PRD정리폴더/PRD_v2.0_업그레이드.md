## 2. v1.0 기존 기능 요약

### 2-1. 핵심 포스트 노출판 (/)
- 9개 조직 × 승계 매트릭스 (F1 현직자 → F2/F3 후보 → 날개과업)
- 신호등 색상 (빨강/노랑/파란불) 리스크 시각화
- 이름·과업·조직 통합 검색 및 색상 필터
- KPI 카드 (총 포스트 수, 공석, 즉시 리스크)
- 관리자 모드: 셀 편집 (과업명·담당자·신호등)
- 엑셀 업로드/다운로드 매트릭스 일괄 교체

### 2-2. 외부 인재 파이프라인 (/external)
- 5단계 파이프라인 (1차검토→심층면접→최종후보→채용결정→보류)
- 이력서 텍스트 AI 파싱 (이름·직급·스킬·직무분야 자동 추출)
- 외부후보 ↔ 핵심포스트 연결 (`assignedTaskKey`)
- 적합도 점수 + 채용 시나리오 자동 생성

### 2-3. 내부 인재 풀 (/talent)
- 전 직원 프로파일 테이블 (DISC·평가등급·갤럽 강점·어학/수리)
- 등급·HP/SP 필터 + 검색
- 직원 상세: DISC, 3개년 평가, 역량 레이더

### 2-4. 스코어링 엔진 (scoreTalent V8.0)
- 6개 차원 매칭: 앵커유사도(25%)·포스트부합(20%)·역량(20%)·DISC(13%)·강점(12%)·즉시성(10%)
- 판정: "즉시 검토" / "육성 후보" / "후순위/보류"
- 판별 근거·리스크 텍스트 자동 생성

## 3. v2.0 신규 기능

### 3-1. 경영진 대시보드 (/executive)
**목적**: CHRO·인사실장이 경영진에게 보고할 수 있는 1페이지 인재 리스크 요약

**구성요소**:
| 영역 | 내용 |
|------|------|
| 핵심 지표 4개 | 조직건강도 점수 / 승계준비율% / 즉시리스크 조직수 / 전체 빨간불 셀 수 |
| 조직별 리스크 테이블 | 조직명·건강도·승계여부·비즈니스 영향 1줄 요약 |
| 페르소나 인사이트 | 가상 CHO·HRBP 관점 핵심 액션 코멘트 3개 |
| v4.0 예고 배너 | 인재 교체 시뮬레이터 링크 |

**조직 건강도 계산 로직**:
```
건강도 = (해당 조직 빨간불 아닌 셀 수) / (전체 활성 셀 수) × 100
- 75점 이상: 안정 (초록)
- 50~74점: 주의 (노랑)
- 50점 미만: 위험 (빨강)
```

**승계 준비율**:
```
승계 준비율 = F2/F3 중 yellow 이상인 후계자가 지정된 조직 수 / 전체 조직 수 × 100
```

**수용 기준**:
- [ ] 4개 핵심 지표 카드가 매트릭스 데이터 기준으로 실시간 계산됨
- [ ] 조직별 테이블에서 건강도·승계여부·비즈니스 영향이 모두 표시됨
- [ ] 페르소나 인사이트 3개 상시 노출
- [ ] v4.0 시뮬레이터 링크 클릭 시 /simulator로 이동

### 3-2. 인재 교체 시뮬레이터 (/simulator) — v4.0 Preview
**목적**: "이 포스트에 이 사람을 넣으면 리스크가 어떻게 바뀌는가?" 에 대한 데이터 기반 답변

**구성요소**:
| 영역 | 내용 |
|------|------|
| 포스트 선택 리스트 | 매트릭스의 모든 current + wings 포스트 목록 (리스크 레벨 배지) |
| 추천 후보 리스트 | scoreTalent V8.0 기반 상위 5명 (현직자 제외) |
| 시뮬레이션 결과 | 리스크 Before → After 시각화 + 비즈니스 영향 분석 |
| 스코어 상세 | 6개 차원 점수 바 차트 |

**리스크 레벨 매핑**:
```
포스트 색상 → 현재 리스크:  red=HIGH / yellow=MEDIUM / blue,green=LOW
후보 종합점수 → 예상 리스크: ≥60=LOW / 45~59=MEDIUM / <45=HIGH
```

**비즈니스 영향 시나리오 생성 기준**:
| 후보 점수 | 배치 판정 | 권고 액션 |
|-----------|-----------|-----------|
| 60점 이상 | 즉시 배치 가능 | 1개월 내 실행, 공백 최소화 |
| 45~59점 | 육성 후 배치 | 3~6개월 코칭·OJT 병행 |
| 45점 미만 | 외부 채용 검토 | 현직자 유지 또는 임시 배치 |

**수용 기준**:
- [ ] 포스트 선택 시 해당 포스트 프로파일 기준 추천 후보 5명 자동 랭킹
- [ ] 후보 선택 시 Before/After 리스크 레벨 시각화
- [ ] 비즈니스 영향 3개 항목 자동 생성
- [ ] 예상 실행 기간 표시
