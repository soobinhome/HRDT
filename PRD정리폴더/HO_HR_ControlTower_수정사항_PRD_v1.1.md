# HO HR Talent Control Tower — 수정사항 PRD v1.1

> 기준 문서: `HO_HR_Talent_ControlTower_PRD_v1.0.md`
> 작성일: 2026-05-29 · 대상: 핵심 포스트 노출판(홈 대시보드)
> 본 문서는 v1.0 구현 이후 사용자 피드백에 따른 UI/기능 개선 사항을 정의한다.

---

## 1. 배경 / 목적

현재 홈 대시보드의 매트릭스는 각 포스트가 "카드" 형태로 표시된다. 사용자 피드백:

- 카드 크기·정렬이 제각각이라 표처럼 정돈돼 보이지 않는다.
- 카드 내 여백이 많고 정작 핵심 정보(이름)는 작아 비어 보인다.
- 신호등 색이 작은 점/좌측 액센트로만 표기돼 한눈에 상태가 안 들어온다.
- 샘플 데이터를 직접 엑셀로 넣어보고 싶다.

목적: **히트맵처럼 한눈에 읽히는 정돈된 매트릭스 + 엑셀로 데이터를 손쉽게 갈아끼우는 워크플로** 제공.

---

## 2. 변경 범위 (4개 항목)

### 2-1. 매트릭스 셀 정렬 정돈 (표 형태)

- 모든 셀을 동일한 크기·구조로 통일해 행/열이 표처럼 딱 맞게 정렬.
- 셀 내부 레이아웃 고정: (상단) 과업명 → (본문) 담당자 이름 → (하단) 직급·나이 메타.
- 고정 최소 높이로 행 높이 균일화. 불필요한 여백 축소.

### 2-2. 카드 → 밀도 높은 셀

- "카드 + 그림자 + 큰 여백" 느낌을 제거하고, 색으로 채워진 밀도 높은 셀(타일)로 전환.
- **담당자 이름을 가장 크고 굵게** 표시(가독성 우선). 과업명은 보조 텍스트.

### 2-3. 신호등 = 셀 전체 색 + 대비 텍스트

- 좌측 액센트/작은 점 대신 **셀 배경 전체를 신호등 색으로** 채운다.
- 텍스트는 배경 명도에 맞춰 흰색 또는 검정으로 대비 확보.
  - 빨간불(red) → 진한 빨강 배경 + **흰색** 텍스트
  - 노란불(yellow) → 앰버 배경 + **검정(잉크)** 텍스트 (앰버는 밝아 흰색 대비 부족)
  - 파란불(blue) → **초록색** 배경 + **흰색** 텍스트
  - 빈칸(empty) → 옅은 회색 점선 셀 + 흐린 텍스트
- 공석(담당자 없음)은 셀 안에 "공석" 강조 표기.

### 2-4. 신호 상태 3단계로 통합 (의사결정 반영)

- 기존 5값(red/yellow/blue/green/empty) 중 **초록불(green)을 별도 상태에서 제거**.
- 화면에 노출되는 신호는 **빨간불 / 노란불 / 파란불** 3가지 (+ 빈칸).
- **파란불은 워딩은 "파란불" 유지, 실제 색상은 초록색**으로 렌더.
- 편집 모달의 색상 선택지에서 초록불 제거. KPI 카드의 "파란불"도 초록색으로 표기.
- 데이터 타입(PostColor)의 green 멤버는 호환을 위해 남기되 신규 입력/선택에서는 사용하지 않는다.

### 2-5. 후계자 명칭 변경: F2 / F3

- 서열 정의: **F1 = 현직자(본인)**, **F2 = 현직자 바로 다음 서열**, **F3 = 그 다음 서열**.
- 매트릭스 헤더: `현직자(F1)` · `F2` · `F3` · `1번 날개`~`5번 날개`.
- 셀 클릭 시 포지션 라벨도 `…F2` / `…F3`로 표기.

### 2-6. 엑셀 업로드 / 템플릿 다운로드 (의사결정 반영)

홈 대시보드 **맨 아래**에 "엑셀로 데이터 관리" 패널 추가.

- **템플릿 다운로드**: 입력 예시가 채워진 `.xlsx` 파일을 즉시 다운로드.
- **파일 업로드**: 작성한 `.xlsx`를 올리면 파싱 → 미리보기/확인 모달 → **전체 교체**.

**엑셀 양식 (방식 B — 조직 한 줄):**

| 조직 | 현직자(F1) | F2 | F3 | 날개1 | 날개2 | 날개3 | 날개4 | 날개5 |
|------|-----------|----|----|------|------|------|------|------|
| 유통BG | BG 총괄 / 권태오(이사대우/46) / 노란불 | / - / 빨간불 | / - / 빨간불 | SA급 컨텐츠 / 남도경(부장/44) / 노란불 | … | … | … | … |

- 한 칸 형식: `과업명 / 담당자 / 신호등` (슬래시 `/` 구분).
  - 과업명·담당자는 비어도 됨(공석은 담당자 `-` 또는 공란).
  - 신호등 값: `빨간불`·`노란불`·`파란불` (또는 영문 red/yellow/blue). 칸 전체가 비면 빈칸 셀.
- 담당자 형식 권장: `이름(직급/나이)` — 화면 메타 표기에 사용.
- 업로드 시 기존 매트릭스를 통째로 대체(되돌리기: 새로고침 전이라면 "샘플로 초기화" 버튼 제공).

## 3. 비범위 / 가드레일

- 백엔드(Supabase/Claude) 연동은 여전히 미포함(샘플·로컬 persist 유지).
- 업로드 파일은 클라이언트에서만 파싱하고 서버 전송·보관하지 않음(시연 보안 원칙 유지).
- 전원 가상 인물 데이터 원칙 유지. 모든 추천 결과 "참고·검토용" 고지 유지.

- 의존성: `xlsx`(SheetJS) 추가.

## 5. 완료 기준 (AC)

1. 매트릭스가 행/열이 표처럼 정렬되고 셀 크기가 균일하다.
2. 각 셀이 신호등 색으로 가득 차고, 이름이 가장 크게/대비 있게 보인다.
3. 신호는 빨강/노랑/파랑(초록색) 3종 + 빈칸이며, 편집·KPI에 일관 적용된다.
4. 후계자 칸이 F2/F3로 표기된다.
5. 홈 하단에서 예시 엑셀을 받아 작성→업로드하면 매트릭스가 교체된다.
6. `tsc --noEmit` 통과, 홈 화면 정상 렌더.
